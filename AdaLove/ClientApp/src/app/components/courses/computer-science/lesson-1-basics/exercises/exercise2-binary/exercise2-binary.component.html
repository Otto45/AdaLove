The values of all digits are summed (added together) to determine the value of the entire number. Knowing this, we can now break down an entire base-10 number the following way:
17,543 = (1 x 10^4) + (7 x 10^3) + (5 x 10^2) + (4 x 10^1) + (3 x 10^0)
Math Fact: Any number raised to the power of 0 (# ^ 0) always = 1
