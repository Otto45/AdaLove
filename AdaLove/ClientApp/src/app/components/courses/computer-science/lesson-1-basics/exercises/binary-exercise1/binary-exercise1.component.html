Before we get into how binary numbers work, we must understand how number systems work in general. Our normal number system is called the decimal system, and is base 10. This means each digit in a number can have an integer value of 0, 1, 2, 3, 4, 5, 6, 7, 8, or 9. Notice that there are 10 choices for a digit's integer value, matching the base number.
The position of a digit in a number also determines the digit's value, along with its integer value.
value = (integer value) x (base number) ^ (digit's position in number from right to left, starting at 0)
In the number 87, the digit 8's value = 8 x 10^1 = 8 x 10 = 80.
