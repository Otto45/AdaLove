We're now ready to apply this number system knowledge to learn binary!
Guess what? The only new thing you need to know to understand binary is that it's base-2 instead of base-10. This means there are only two integer values a digit can have, 1 and 0. Using our knowledge of how number systems work, we can represent the number "13" in binary like this:
1101 = (1 x 2^3) + (1 x 2^2) + (0 x 2^1) + (1 x 2^0) = 13
Notice each digit in a binary number is a power of 2 (like each digit in a base-10 number is a power of 10). Since binary is a foundational pillar of computer science, you should remember the first few powers of 2, as you'll see them all the time:
0, 2, 4, 8, 16, 32, 64, 128, 256, 512, 1024
