Before we get into how binary numbers work, we must understand how number systems work in general. Our normal number system is called the decimal system, and is base 10. This means each digit in a number can have an integer value of 0, 1, 2, 3, 4, 5, 6, 7, 8, or 9. Notice that there are 10 choices for a digit's value, matching the base number.
The position of a digit in a number also determines the digit's value, along with its integer value. value = (integer value) x (base number) ^ (digit's position in number from right to left, starting at 0)
In the number 87, the digit 8's value = 8 x 10^1 = 8 x 10 = 80.
The values of all digits are summed (added together) to determine the value of the number the digits represent. Knowing this, we can break down a base-10 number the following way:
17,543 = (1 x 10^4) + (7 x 10^3) + (5 x 10^2) + (4 x 10^1) + (3 x 10^0)
With this understanding, we are ready to learn the binary number system.
